# Student Performance & Lifestyle Analysis
## Overview
An end-to-end data analysis project exploring how lifestyle and behavioral factors influence student academic performance. Built using a Kaggle dataset of 2,000 student records across 9 variables.
## Key Finding
Study hours and stress level were the strongest predictors of student grades. A linear regression model built on lifestyle factors alone achieved an R² of 0.55.
## Features

Data cleaning and validation with pandas
Exploratory data analysis with histograms, boxplots, and skewness
Correlation heatmap across all numeric variables
Linear regression model with train/test split

## Tools
Python - pandas, NumPy, Matplotlib, Seaborn, scikit-learn
